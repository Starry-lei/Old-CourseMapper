# Course Mapper


## Development setup

### Requirements

1. MongoDB (installed and running)
2. NodeJS and npm (installed)

### Usage

```sh
# Install tools globally
sudo npm install -g bower gulp grunt nodemon

# Install JS modules
sudo npm install -g npm-run-all
bower install

# Build front-end JS
grunt

# Run Mocha tests
npm test

# Start web app
./bin/www
or nodemon
```
